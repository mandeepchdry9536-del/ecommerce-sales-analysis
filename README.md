# E-commerce Sales Data Analysis

## 📌 Project Overview
This project analyzes an e-commerce sales dataset to identify trends, patterns, and business insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Analysis
- Monthly sales trends
- Top-selling products
- Profit by category
- Region-wise performance
- Loss-making areas

## 💡 Insights
- Sales peak during festive months
- Technology category is most profitable
- Some categories need pricing optimization

## 🚀 Recommendations
- Increase marketing during peak months
- Focus on high-profit categories
- Improve underperforming regions
